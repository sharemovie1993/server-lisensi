🔧 Inisialisasi Otomatis Infrastruktur VPN VPS

Siapkan VPS baru secara otomatis: install WireGuard, Nginx, Certbot, IP Forwarding, dan buat skrip tunnel otomatis. Jalankan ini sekali saja pada VPS yang baru di-deploy.

📋 Daftar Komponen yang Akan Dipasang & Dikonfigurasi:

  • Pemasangan Paket Sistem: Wireguard, Nginx, Certbot, Python3-Certbot-Nginx via APT
  • Aktivasi Forwarding IP: Konfigurasi net.ipv4.ip_forward = 1 di Kernel Linux
  • Konfigurasi Awal Wireguard: Membuat key pasang peer dan mengaktifkan interface wg0 (Port: 51820 UDP)
  • Pembuatan Skrip Otomasi: Menulis berkas pembantu /usr/local/bin/add-wg-peer.sh
  • Isolasi Firewall Klien: Menerapkan aturan iptables isolasi peer agar sesama tenant terisolasi penuh

*Proses ini memakan waktu sekitar 1-2 menit. Halaman web akan menampilkan log progress secara real-time. Pastikan VPS terhubung ke internet.
